#  Tasty Restaurant Website

##  Project Description
Tasty Restaurant Website is a simple and responsive restaurant website developed using HTML, CSS, and JavaScript. It allows users to explore the restaurant menu, view food items, and experience an attractive user interface.

##  Features
- Responsive Homepage
- Navigation Bar
- Food Menu
- About Section
- Contact Section
- Attractive UI Design
- Mobile Friendly Layout

##  Technologies Used
- HTML5
- CSS3
- JavaScript

##  Project Structure
```
Restaurant-Website/
│── index.html
│── menu.html
│── about.html
│── contact.html
|── cart.html
│── style.css
│── script.js
└── README.md
```

##  How to Run
1. Download or Clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it using Live Server in VS Code.
